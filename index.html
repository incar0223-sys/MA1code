🎯 사용 가이드
  1. 아래 입력창에 사번 7자리 입력 후 검색 버튼 클릭 또는 ENTER
  2. 사번 조회 후 보험사 칸에 일부만 입력해 좁히기(ENTER로 적용). 보험사 초기화는 해당 사번의 전체 목록으로 되돌립니다.
  3. 사번 초기화는 입력·목록을 비우고 데이터를 다시 불러옵니다.
  4. 조회된 결과에서 📅 버튼 클릭 → 해당 보험사·접수차수에 맞는 인사일정 확인 (서류마감 / URL발송 / 전자동의마감 / 위촉일). 표 헤더 일정 ⓘ 안내 참고
  5. 비고 열 ⓘ: 기존 위촉차수 결과 표시(실시간 연동 아님, 참고용)
  6. 보험사 전체일정 확인 버튼 클릭 → 전체 보험사 일정 한눈에 확인
⚠️ 필독 — 주의사항
🚫 주의사항 필독 — FA배포 금지
  1. 위촉접수 조회리스트는 센터원과 파트너 총무와의 원활한 업무진행을 위한 자료입니다.
  2. 확정된 리스트이며 기존 차수에 추가접수를 위한 자료가 아닙니다. 접수요청 불가!!
  3. 접수차수를 확인후 위촉예정일을 조회할 수 있습니다. 당일기준 약 2~3개월 전 접수 리스트 조회 가능합니다.
  4. 업데이트는 상시 되고있으며 실시간 접수목록이 반영되지는 않습니다.
  5. 센터내에서 리스트가 확정된 익일 or 익익일 확인 가능
  6. URL 발송일 및 위촉 일정은 보험사의 사정에 의하여 지연될 수 있습니다.
  7. 재 위촉 진행시 생명보험 협회등록 당시 정보와 변동이 있을경우 꼭 파트너담당 센터원에게 공유 해주셔야 합니다.
    ex)협회등록이후 IIMS 연락처 변경 / 위촉지사 변경
🚫 접수리스트에 없는 경우

  • 보험사/MA센터/영업지원팀 전체 협의 후 진행하는 경우
